## 🛠️ 故障排查

### 页面打不开?

```bash
# 检查服务是否运行
ps aux | grep vite

# 如未运行,重新启动
cd ~/market-supervision-system/frontend
npm run dev
```

### 样式不正常?

```bash
# 清除缓存重新构建
rm -rf dist .vite
npm run dev
```

### TypeScript错误?

```bash
# 重新安装依赖
rm -rf node_modules package-lock.json
npm install
```
