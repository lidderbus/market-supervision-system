## 💡 文档使用技巧

### 1. Markdown渲染
```bash
# 在VS Code中打开
code ~/market-supervision-system

# 预览Markdown (Cmd+Shift+V 或 Ctrl+Shift+V)
```

### 2. 快速搜索
```bash
# 在所有文档中搜索关键词
cd ~/market-supervision-system
grep -r "关键词" *.md docs/*.md
```

### 3. 打印输出
```bash
# 转换为PDF (需要pandoc)
pandoc FINAL_REPORT.md -o FINAL_REPORT.pdf
```
